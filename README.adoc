= tyk-sync-foo

A dashboard plugin

This repository contains a Go-based plugin that is distributed via releases.

== Development

=== Prerequisites

* Go 1.20 or higher
* Make

=== Building

To build the plugin:

[source,bash]
----
make build
----

The compiled plugin will be available in the `dist` directory.

=== Testing

Run the tests:

[source,bash]
----
make test
----
